HTML sta per Hyper Text Markup Language. Viene utilizzato per creare pagine Web e applicazioni. Questo linguaggio è facilmente comprensibile dall'utente ed è anche modificabile. In realtà è un linguaggio Markup, quindi fornisce un modo flessibile per progettare il pagine web insieme al testo.

Il file HTML è composto da diversi elementi. Un elemento è una raccolta di tag di inizio, tag di fine, attributi e il testo tra di essi.
