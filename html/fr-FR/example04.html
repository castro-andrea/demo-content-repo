HTML signifie Hyper Text Markup Language.
Il est utilisé pour créer des pages Web et des applications. Ce langage est facilement compréhensible par l'utilisateur et également modifiable. Il s'agit en fait d'un langage de balisage, il offre donc un moyen flexible de concevoir le pages Web avec le texte. TEST : AJOUT D'UN NOUVEAU TEXTE DANS CE PARAGRAPHE

Le fichier HTML est composé de différents éléments. Un élément est une collection de balise de début, balise de fin, attributs et le texte entre eux.
